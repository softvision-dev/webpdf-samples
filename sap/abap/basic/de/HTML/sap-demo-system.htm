SAP Entwickler-System (als Demo-System)

 

Für die nachfolgenden Beispiele können Sie entweder Ihr vorhandenes SAP-System oder die Entwickler-Version des "ABAP application server on ASE 16.0" verwenden.

 

Hinweis: Dieses Dokument verwendet für die nachfolgenden Beispiele ein SAP Entwicklersystem als Demo-System, das gemäß dem Blog-Artikel https://blogs.sap.com/2019/07/01/as-abap-752-sp04-developer-edition-to-download/ eingerichtet wurde. Der Artikel beschreibt ausführlich die die Einrichtung einer solchen Demo-Umgebung.

 

Diese Installation stellt die folgenden Komponenten zur Verfügung:

 

•SAP AS ABAP 7.52 SP04

•SAP GUI for the Java 7.5 and SAP GUI for Windows 7.50

•SAP Sybase ASE 16.0 SP02

 

Mit diesen Komponenten können Sie die nachfolgenden Beispiele nachvollziehen.

 

Eine entsprechende Anleitung zur Installation eines SAP-Systems steht für VirtualBox und VMware zur Verfügung:

 

•Installing SAP AS ABAP 752 SP04 on Linux – Virtual Box

•Installing SAP AS ABAP 7.52 SP04 on Linux – VMWare

 

Für diese Dokumentation wurde ein Demo-System nach der Anleitung "Installing SAP AS ABAP 752 SP04 on Linux – Virtual Box" erstellt.
